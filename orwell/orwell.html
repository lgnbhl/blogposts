A Sentiment Analysis by Chapter

Sentiment words used in 1984 are mostly describing Winstonâ€™s emotions. So it makes sense to try a sentiment analysis. Letâ€™s take the Text Mining with Râ€™s sentiment analysis approach, defining a sentiment score for each chapter i of the book:

scorei = positivei - negativei

where positivei is the number of â€œpositiveâ€ words in a chapter, and negativei the number of â€œnegativeâ€ words in a chapter. For instance, if we have 100 positive words and 60 negative words in chapter 1, we get a score of 40.

First, we need to add a chapter variable in our dataset.

bigBro_2 <- data_frame(text = read_lines("input/text_1984.csv")) %>%
  mutate(chapter = cumsum(str_detect(text, regex("^chapter [\\digit]", 
                                                 ignore_case = TRUE)))) %>%
  unnest_tokens(word, text, format = "text")

Now, letâ€™s prepare our dataset for the sentiment analysis.

library(tidyr)

bigBro_sent <- bigBro_2 %>%
  inner_join(get_sentiments("bing")) %>%
  anti_join(stop_words) %>%

  mutate(sentiment = positive - negative)
